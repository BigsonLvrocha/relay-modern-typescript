## Relay Modern Typescript

Starter of Relay Modern with Typescript using Webpack on Web

### Setup

- Install dependencies
```bash
yarn install
```

- Run relay-compiler to generate GraphQL artifacts (static queries)
```bash
yarn relay
```

- Start web app
```bash
yarn start
```

Open web app at: 

http://[::]:55555 (ipv6) or
http://localhost:55555 (ipv4)
